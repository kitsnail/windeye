# Value Assessment Guide

## Assessment Dimensions

| Dimension | Question | Weight |
|-----------|----------|--------|
| **Decision Impact** | Does this information affect decisions? | 40% |
| **Action Trigger** | Does this information trigger actions? | 30% |
| **Insight Output** | Does this information produce insights? | 30% |

## Assessment Flow

```
New information → Initial rating: low
        ↓
Continued tracking → Value discovered → Upgrade to medium/high
        ↓
Regular review → No value → Downgrade/archive
```

## Value Levels

### High (🔥)

**Criteria**:
- Directly affects decisions
- Immediate action required
- Clear impact path

**Treatment**:
- Priority tracking
- Regular review
- Link to conclusions

### Medium (⭐)

**Criteria**:
- Triggers action or insight
- Potential impact
- Worth continued attention

**Treatment**:
- Continued attention
- Relationship analysis
- Monitor for upgrade

### Low (📌)

**Criteria**:
- Potential value
- Uncertain impact
- Needs observation

**Treatment**:
- Periodic review
- Decide keep/remove

## Assessment Questions

### For Decision Impact (40%)

1. Does this change my understanding of the situation?
2. Does this affect my priorities?
3. Does this change my action plan?
4. Does this influence resource allocation?

### For Action Trigger (30%)

1. Does this require immediate action?
2. Does this create new opportunities?
3. Does this reduce risks?
4. Does this enable new capabilities?

### For Insight Output (30%)

1. Does this reveal new patterns?
2. Does this challenge assumptions?
3. Does this connect previously separate ideas?
4. Does this simplify complex situations?

## Scoring

```
Score = (Decision Impact × 0.4) + (Action Trigger × 0.3) + (Insight Output × 0.3)

Each dimension: 0-1 scale

Result:
- Score ≥ 0.7 → High
- Score 0.4-0.7 → Medium
- Score < 0.4 → Low
```
