# 信号模板

## 每日信号格式

```markdown
# 每日信号 YYYY-MM-DD

> 生成时间：YYYY-MM-DD HH:MM
> 数据来源：GitHub API / Web Search
> 关注点数量：N 个
> 发现数量：M 个

---

## 🔍 今日发现

### 1. [发现标题] 🔥/⭐

**发布时间**：YYYY-MM-DD
**来源**：[关注点名称]
**链接**：[URL]

**核心更新**：
- 要点 1
- 要点 2

**影响**：[对用户的潜在影响]

---

### 2. [发现标题]

...

---

## 📊 关注点状态

| 关注点 | 检查状态 | 发现 | 数据来源 |
|--------|:--------:|:----:|:--------:|
| xxx | ✅ | 1 条 | GitHub API |
| yyy | ⚠️ | - | 访问失败 |

---

## 💡 洞察

[从发现中提炼的洞察]

---

## 🎯 建议行动

1. [行动建议 1]
2. [行动建议 2]

---

*由 Daily Signal 自动生成*
```

---

## 信号片段格式

每个 Focus 的信号片段：

```markdown
### [Focus 标题] - 新发现

**版本**：vX.Y.Z
**发布时间**：YYYY-MM-DD

**更新内容**：
- 新功能
- 破坏性变更
- 废弃通知

**来源**：[URL]
```

## 状态标记

| 标记 | 含义 |
|:----:|------|
| ✅ | 检查成功 |
| ⚠️ | 访问失败 |
| 🔥 | 高价值发现 |
| ⭐ | 中等价值 |
| 📌 | 低价值 |
